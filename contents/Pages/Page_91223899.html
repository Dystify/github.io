6.2 System Fonts

Describes NX system font specifications and cautions.

Basic Information

The system fonts are as follows.

For Japan/North America/Europe

A font that mainly includes characters of languages for Japan, Europe and North America.

  • Font name: nintendo_udsg-r_std_003
  • 7770 characters

For Hangul

A font that mainly includes characters for languages that use Hangul.

  • Font name: nintendo_udsg-r_ko_003
  • 17708 characters

For Simplified Chinese

A font that mainly includes characters for languages that use simplified Chinese.

  • Font name: nintendo_udsg-r_org_zh-cn_003 & nintendo_udsg-r_ext_zh-cn_003
    • In the font data specifications, system fonts and supplementary fonts are used in combination.
      Use NintendoSDK features to set the priority for the read fonts.
      The priority is (high) supplementary > (low) system.
    • System font: nintendo_udsg-r_org_zh-cn_003
    • Supplementary font: nintendo_udsg-r_ext_zh-cn_003 (The character count is comparatively less than the system. Characters that are not included in this font are read from the system font.)
  • 29,138 characters

For Traditional Chinese

A font that mainly includes characters for languages that use traditional Chinese.

  • Font name: nintendo_udjxh-db_zh-tw_003
  • 14549 characters

For Nintendo Extended Characters

An outline font of encoded pictograms made within Nintendo that we want to use as a system feature font.

  • Font names: nintendo_ext_003, which includes the extended characters for the Nintendo 3DS and the Wii U, and nintendo_ext_LE_003, which does not.

Supported Languages

The scopes for Japan/North America/Europe, Hangul, simplified Chinese and traditional Chinese are common encoded characters (common) and uncommon encoded characters (only).
The supported languages for each scope are as follows.

  • Common encoded characters (common): characters included in and shared by 4 fonts. ASCII, European and North American characters, kana characters, and generic symbols.
  • Uncommon encoded characters (only): characters that are unique to the individual fonts. Chinese characters for Asian languages. Separated within Unicode specifications.

Notes

  • There is no variation in weight and typeface in regard to memory.
  • Can be made italic with the rendering engine. Boldface characters are possible, but using them consumes memory.
  • There is no support for pictograms.
