3.4 My Page

A system application that brings together functionality and data specific to the user.

Includes features like My Information, Friend List, and User Settings.

My Information

The following options are available.

  • Check the profiles and the presence of your own user account.

Friend List

The following options are available.

  • Check the profiles and the presence of friends, set favorites, and delete friends.
  • Send, receive, accept, and delete friend requests.
  • Check the Player History.
  • Manage the blocked-user list.

Functionality is available for jumping to some of these features from within the application and for calling some of the involved processes. Using that functionality can reduce the implementation load for the UI and processing specific to friend features.

For more information about friend features, see Friends and Use From Applications within the Account Guide.

User Settings

Displays an Advanced Settings screen for each user configured within the system settings.

The available options are identical to those in the relevant page within the system settings.
