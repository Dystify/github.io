3.3 Captures and Albums

Screenshots captured using the Capture button are saved to the Album.

This section defines screenshot data and describes how to capture, view, edit, and post it.

Screenshot Data

Table 3-1 Screenshot Specifications
Item Description
Format JPEG
Size 1280 × 720 pixels (identical to console LCD screen resolution)
Capacity 500 KB or less (TBD)
Supplemental Information Capture date and time, title identifier, signature, and other data stored to the EXIF region.

Capturing

You can take application screenshots by pressing the Capture button. During capture, an overlay notification appears on the upper left of the screen for the application. At the same time, a sound effect is played.

Screenshots are saved either to the system memory or to an SD card. The user can select the save location using the system settings. By default, the SD card is used. Captured data will be saved to the system memory when no SD card is inserted, even when the option to use the SD card is selected.

The restrictions on the number of screenshots for each location are as follows. Saving will no longer be possible when the number restriction is met.

Table 3-2 Restrictions on Number of Screenshots
Save Destination Number of Slides
System Memory 1000
SD Card 10000
Info

The ways to use video capture and application-controlled auto-capture are being considered separately.

Blocking Captures

The following method is under discussion as a means to block captures.

  • A setting for blocking captures, to block across specific software, will be available for the META settings file needed to create ROMs.
  • Offer capture blocking APIs, to block certain scenes within software.

Viewing

To view screenshots in order starting from the most recent captures, from the HOME Menu, select Albums. Functionality to filter display based on titles is available in case there are a lot of screenshots.

A signature check will be performed on screenshots. They will not be displayed if it has been determined that screenshots were modified using a device other than the NX.

Editing

You can write text on screenshots.

In Albums, select the screenshot to edit. You can then use the text entry option from within the Edit menu. Text size, position, and color can all be customized.

Info

Separate discussions are underway to implement editing features beyond text entry.

Posting

Screenshots can be posted to social media. The social media to target are currently under discussion. It is also possible that social media options will be added or removed after sales of the NX begin.

Select the screenshot to post within Albums. Click the Post button. The account management applet will start. Select the user. The post screen will appear. Comments can be entered and posted along with the screenshot.

Note

Currently, it is not possible to post from NX development hardware.
