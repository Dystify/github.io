5.4.1 Overlay Notifications

This feature displays urgent information on the screen when system power is on (including when applications are running). It appears in the upper-left part of the screen.

  • Volume change
  • Screen capture
  • System (low remaining battery life, start charging, download completed)
  • Friend Presence
  • Time remaining on the play timer (when there is little time left) or an indication that the set time has been exceeded.
  • Controller (low remaining battery life, disconnected)

For more information about the play timer feature, see 5.5 Setting and Checking From Smart Device Apps.
