3.2 Nintendo eShop

This functionality is for purchasing downloadable content, applications, and service items for sale through Nintendo eShop.

The following types of content are sold.

  • Downloadable applications made especially for the NX
  • Downloadable content used within applications

In addition, functionality for distributing screenshots, video, and demo versions is also available.

Regions and Sales Countries

NX specifications will not use region locks. As a result, it will be possible to play any downloadable software on any NX.
To sell downloadable software in multiple countries, however, you would need to configure the settings for each country because Nintendo eShop is administered at the country level.
Players can use Nintendo eShop for the country indicated by each player as a residence when adding the Nintendo Account. Players will then be able to purchase downloadable software and content sold in that country.

Info

For more information about applying for and configuring sales on Nintendo eShop,see the Master ROM Submission Guidelines, set to be released at a later date.
