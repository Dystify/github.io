6.1 Mii

Mii characters allow characters with the user's face to appear in applications as avatars of the user. The NX can save up to 100 Mii characters for each console.

Note that, to register Mii characters to applications, the Face Library is required. Create Mii characters using the Create or Edit Mii Applet.

Differences Between Mii Characters on the 3DS and Wii U and Mii Characters on NX

Parameters to Expand

The number of colors for hair, eyebrows, eyes, facial hair, glasses, mouths, and skin.

Parameters to Remove

  • Creator's Name
  • Favorites
  • Birthday
  • Public (The Mii Encounters feature will go away.)
  • Copying (Anyone can edit Mii characters.)
