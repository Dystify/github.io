1 Introduction

Objectives and Content

This documentation describes the specifications for system features and describes how to use them in applications, presenting more detailed information than in the NX Overview.

Intended Audience

This documentation is intended for planners and developers of NX applications.

When to Read This Document

Read this documentation when you are planning and developing NX applications.

Related Documents

The NX Overview also presents a simple overview of system features. Before reading this documentation, read the documentation about system features in the NX Overview.
