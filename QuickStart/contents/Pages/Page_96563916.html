2 SDEV Operating Environment

The following environment is required for SDEV setup and the execution of NX applications.

Operating System
  • Microsoft Windows 7 Professional (64-bit)
  • Microsoft Windows 10 Pro (64-bit)
Info

An account with administrator rights is required.

.NET Framework
  • Microsoft .NET Framework 4.6.1 or later
Info

If not already installed, download and install from the following link.

https://www.microsoft.com/en-us/download/details.aspx?id=49981

SDEV
  • SDEV 1.6
  • SDEV 1.8
  • SDEV PreMP1 (1.9)
  • SDEV PreMP2

In addition, the following environment is required for developing applications with the NintendoSDK.

Operating System
  • Microsoft Windows 7 Professional (64-bit)
  • Microsoft Windows 10 Pro (64-bit)
Info

An account with administrator rights is required.

.NET Framework

  • Microsoft .NET Framework 4.6.1 or later

Info

If not already installed, download and install from the following link.

https://www.microsoft.com/en-us/download/details.aspx?id=49981

Visual Studio
  • Microsoft Visual Studio 2013 Professional Edition Update 5
  • Microsoft Visual Studio 2015 Professional Edition Update 3
Info

Required in order to build and debug NX applications.
Not necessary if not building and debugging the application.
Info

To use the NintendoSDK, you must install the Visual C++ redistributable packages.
They will be automatically installed when you install the NintendoSDK from NDI.
If you are not using NDI, use one of the following installers to install the packages.

Package Type Installer Path
Visual C++ Redistributable Packages for Visual Studio 2012 NintendoSDK\Installers\VcRuntime\v110\
  • vcredist_x64.exe
  • vcredist_x86.exe
Visual C++ Redistributable Packages for Visual Studio 2013 NintendoSDK\Installers\VcRuntime\v120\
  • vcredist_x64.exe
  • vcredist_x86.exe
Visual C++ Redistributable Packages for Visual Studio 2015 NintendoSDK\Installers\VcRuntime\v140\
  • vc_redist.x64.exe
  • vc_redist.x86.exe
