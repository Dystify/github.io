5.3 Library Applets

Library applets are small, modularized applications that are created for a specific purpose. You can call a library applet while playing an NX application, and then return to the application when the applet closes.

However, because this uses a dedicated memory region for library applets, only one library applet can run at a time.

This section describes the following library applets.

  • 5.3.2 Controller Support Applet
  • 5.3.3 Software Keyboard Applet
